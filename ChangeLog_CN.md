# FoxGet Package Manager

FoxGet 是 VFP 的软件包管理器，类似于 .NET 的 NuGet 软件包管理器。

## 发布

### 2024-07-21

* 添加了 VFPX Framework 安装程序。
* 更新至最新 VFPX Framework 版本。

### 2024-01-30

* 添加了 fpCefSharp、Format 和 VFP2C32 的安装程序。
* 实施 VFPX Framework。

### 2024-01-07

* 添加了对 MyPackages(自定义软件包列表)的支持，这对测试或私人软件包非常有用。
* 添加了对本地安装程序 PRG 的支持。
* 添加了对复制提取文件子目录的支持。
* 修复了一些小错误。

### 2023-12-28

* 初次发布
